The Game

Über uno is an umbrella term for a set of card games played with Uno cards. The game started about 15 years ago and the rules and variations of the game have been evolving since. When a new associate joins the plant, he/she has a very steep learning curve to catch up with the variations of the game that have evolved over the years. This website is intended to be an aid to hasten the process of learning the game.
